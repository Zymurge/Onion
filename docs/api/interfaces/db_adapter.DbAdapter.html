    Interface DbAdapter

    Data Access Layer interface for Onion game persistence.

    Provides a clean abstraction over storage backends (in-memory, PostgreSQL, etc.). All operations are async to support both fast in-memory implementations and slower database-backed ones.

    Implementations must be thread-safe for concurrent requests.
